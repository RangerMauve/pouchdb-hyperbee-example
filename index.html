PouchDB Hyperbee Example

What is this?

This is a demo of pouchdb-adapter-hyperbee, it uses the p2p database Hyperbee to store changes and enables Pouch to load data as-needed from remote databases instead of needing to fully replicate.

This works particularly well when you set up indexes using Pouch's Mango Queries
